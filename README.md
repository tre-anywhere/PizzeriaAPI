# Pizzeria API
version 1.1.0

This is an ExpressJS Pizzeria API. Call me for a pie anytime!

## Getting Started

1. Starts a web server on ( `localhost:3000` )
2. Serves an image on ( `http://localhost:3000/` )
3. run with `npm run devStart`
